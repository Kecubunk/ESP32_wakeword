# ESP32_wakeword
Wakeword For ESP32 using INMP4441

# INMP441 Pin Connection :
I2S_MIC_SERIAL_CLOCK GPIO_NUM_14  -> SCK   </br> 
I2S_MIC_LEFT_RIGHT_CLOCK GPIO_NUM_15 -> WS </br> 
I2S_MIC_SERIAL_DATA GPIO_NUM_32 -> SD </br>

# max98357A Connection pin
I2S_DIN       25
I2S_BCLK      27
I2S_LRC       26
